# 新手村玩法循环

最后更新：2026-06-21

本文档是第一章当前玩法基线。

## 目标

制作一个 20-30 分钟的首次体验循环，教授移动、NPC 交互、词汇挑战、奖励、装备和第一次 Boss 遭遇。

玩家应该感受到：

- “我知道要去哪里。”
- “我明白为什么要学习这些词。”
- “RPG 系统正在奖励我的词汇进度。”
- “村庄看起来像真正的游戏，而不是原型。”

## 核心循环

```text
探索村庄
-> 注意到附近 NPC / 目标提示
-> 开始对话
-> 完成词汇挑战
-> 获得书页 / 物品 / 数值奖励
-> 看到 HUD 和背包更新
-> 收集 3 页后解锁森林入口
-> 挑战 Word Imp
-> 看到战斗结算和下一个目的地
```

## 开场状态

- 玩家以 1 级战士身份开始。
- 玩家只有基础新手装备。
- 书页：`0/3`。
- 森林入口封闭。
- 三个支线 NPC 可交互。

## 支线

### 图书馆支线

目的：

- 教授单词释义识别。

挑战：

- 根据中文含义选择正确英文单词。

奖励：

- 书页 1。
- 少量金币或新手知识奖励。

视觉锚点：

- 图书馆 / 书本主题房屋。
- NPC 站在门口或书摊附近。

### 铁匠支线

目的：

- 教授拼写回忆。

挑战：

- 根据中文含义或视觉提示，输入 / 拼出英文单词。

示例：

- `剑` -> `sword`

奖励：

- 书页 2。
- 新手武器 / 盾牌 / 战斗数值提升。

视觉锚点：

- 熔炉、锤子、铁砧、木箱或铁匠房屋。

### 花园 / 水井支线

目的：

- 教授语境中的单词使用。

挑战：

- 用正确英文单词补全简单句子。

示例：

- `I drink ____.` -> `water`

奖励：

- 书页 3。
- 药水或小型防御奖励。

视觉锚点：

- 花园、水井、植物、南瓜、稻草人或路边村民。

## 森林入口与 Word Imp

在书页达到 `3/3` 之前：

- 森林入口保持关闭。
- 入口反馈告诉玩家还缺几页书页。

书页达到 `3/3` 后：

- Word Imp / Boss 遭遇可用。
- 玩家通过上下文 `挑战` 提示开始战斗。

胜利后：

- 结算界面显示胜利和奖励。
- 森林入口打开，通往下一个地区。

## 学习到 RPG 奖励的映射

| 学习时刻 | RPG 回报 |
| --- | --- |
| 学会 `sword` | 武器或攻击力奖励 |
| 学会 `shield` | 副手 / 防御奖励 |
| 学会 `book` | 书页进度或魔法书剧情解锁 |
| 学会 `water` | 药水 / 恢复奖励 |
| 学会 `friend` / `home` | NPC 信任 / 村庄剧情奖励 |

## 当前非目标

- 完整任务日志。
- 存档系统。
- 多职业。
- 复杂主动技能循环。
- 随机地牢。
- 大规模词汇课程。

## 评审检查清单

- 每条支线都有视觉清楚的 NPC 和地点。
- 每个 NPC 都站在自己的房屋或任务锚点附近。
- 道路引导玩家前往房屋、入口和支线区域。
- 上下文提示只在范围内出现。
- 对话和题目文本中文优先。
- 英文单词只作为学习内容出现。
- 奖励通过 HUD、背包、装备或结算界面可见。
