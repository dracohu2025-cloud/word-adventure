## 章节目标

“书灯村”是玩家第一次理解《万语之书》、词灵、英语第一卷、NPC 支线、书页、装备奖励和 Word Imp 战斗的地方。

这一章的体验目标：

- 让玩家知道：学会词语可以修复世界。
- 让玩家知道：NPC 任务会带来书页、金币、装备或战斗成长。
- 让玩家知道：收集 3 页书页后，可以挑战守在森林入口的 Word Imp。
- 让玩家第一次看到：战斗不是答题卡，而是 RPG 数值和动画的结果。

## 章节一句话

普通孩子被《万语之书》选中，在村民帮助下找回英语第一卷的三页书页，并击败迷失词灵 Word Imp，打开通往低语森林的道路。

## 核心词池

第一章不要超过 12-20 个活跃词。

### 必学词

| 单词 | 中文含义 | 剧情用途 | 系统用途 |
| --- | --- | --- | --- |
| `book` | 书 | 解释《万语之书》和收服词灵 | 图书管理员支线、书页 1 |
| `sword` | 剑 | 唤醒见习勇者的攻击力量 | 铁匠支线、攻击/武器奖励 |
| `shield` | 盾 | 保护主角不被失控词灵伤害 | 园丁支线、防御/盾牌奖励 |

### 支撑词

| 单词 | 中文含义 | 用途 |
| --- | --- | --- |
| `water` | 水 | 后续可用于井边、药水、恢复任务 |
| `home` | 家 | 村庄主题和 NPC 归属感 |
| `friend` | 朋友 | 温暖基调和村民信任 |
| `tree` | 树 | 低语森林过渡 |
| `stone` | 石头 | 地图道具和矿道伏笔 |
| `fire` | 火 | 铁匠炉和后续魔法效果 |
| `forest` | 森林 | 下一张地图目标 |
| `gate` | 门 | 森林入口封印 |
| `help` | 帮助 | NPC 任务动机 |

## 主要角色

| 角色 | 功能 | 当前实现对应 |
| --- | --- | --- |
| 主角 | 普通孩子，被选为见习勇者 | `Player` |
| 《万语之书》 | 世界观解释、书页收集、词灵记录 | 当前可先通过 NPC 台词表达 |
| 图书管理员 | 解释书页和 `book` | `LibraryNPC` |
| 铁匠 | 把 `sword` 和装备成长绑定 | `BlacksmithNPC` |
| 园丁 | 把 `shield` 和保护村庄绑定 | `GardenNPC` |
| Word Imp | 第一只迷失词灵，森林入口守门 Boss | `WordImpBoss` |

## 开场流程

### 场景状态

- 主角出现在村庄道路附近。
- HUD 显示书页 `0/3`。
- 三个 NPC 头顶在主角靠近时出现可交互提示。
- 森林入口暂时封闭。

### 开场叙事

第一版可以不做独立过场，先通过第一个 NPC 或近身提示表达。

推荐由图书管理员承担开场解释：

```text
图书管理员：你终于醒了。
图书管理员：这本书刚才飞到你身边，还自己翻开了。
图书管理员：它叫《万语之书》，记录着世界上许多事物的真名。
图书管理员：现在，英语卷的书页散落到了村子里。
图书管理员：如果不把书页找回来，森林入口的词灵会越来越失控。
```

## 支线一：图书管理员与 `book`

### 任务目的

让玩家第一次理解：

- 英语词是“真名”。
- 学会词语可以修复书页。
- 选择题是最轻量的第一种挑战。

### 交互前台词

```text
图书管理员：魔法书的第一页就在附近。
图书管理员：它上面写着一个最重要的真名：book。
图书管理员：看中文意思，选出正确的英文单词。
```

### 挑战

| 字段 | 内容 |
| --- | --- |
| 任务 ID | `library` |
| 挑战类型 | 释义识别 |
| 题面 | `书` |
| 选项 | `book` / `magic` / `forest` / `sword` |
| 正确答案 | `book` |

### 答对反馈

```text
图书管理员：答对了！
图书管理员：book，就是“书”的英语真名。
图书管理员：这页书回来了。以后你可以用它记录迷失的词灵。
```

### 答错反馈

```text
图书管理员：别急，这页书只是又飞高了一点。
图书管理员：中文是“书”，再看看哪个英文更像它的真名。
```

### 奖励

- 书页 +1。
- 小额金币或基础经验。
- 后续可解锁《万语之书》图鉴入口。

## 支线二：铁匠与 `sword`

### 任务目的

让玩家理解：

- 单词学习能转化成战斗成长。
- `sword` 与攻击、武器、勇者身份绑定。
- 拼写回忆比释义识别更进一步。

### 交互前台词

```text
铁匠：炉子上的铭文全乱了。
铁匠：没有正确的真名，剑就只是一块冷铁。
铁匠：你能拼出“剑”的英语真名吗？
```

### 挑战

| 字段 | 内容 |
| --- | --- |
| 任务 ID | `blacksmith` |
| 挑战类型 | 拼写 |
| 题面 | `剑` |
| 正确答案 | `sword` |

### 答对反馈

```text
铁匠：拼得漂亮！
铁匠：sword，这就是剑的真名。
铁匠：拿着它。真正的勇者不是一开始就强，而是愿意一次次练习。
```

### 答错反馈

```text
铁匠：字母还没敲准，剑还没有醒。
铁匠：慢慢来，先记住开头是 s。
```

### 奖励

- 书页 +1。
- 训练木剑或攻击成长奖励。
- 若实现奖励入包，则进入背包并可右键装备。

## 支线三：园丁与 `shield`

### 任务目的

让玩家理解：

- 词语可以出现在句子里。
- `shield` 与保护、防御和村庄安全绑定。
- 学习不是孤立词表，而是在语境中理解。

### 交互前台词

```text
园丁：花园的护栏挡不住失控的词灵。
园丁：有些词不是用来攻击的，而是用来保护大家。
园丁：把合适的英文单词填进句子里吧。
```

### 挑战

| 字段 | 内容 |
| --- | --- |
| 任务 ID | `garden` |
| 挑战类型 | 语境选择 |
| 题面 | `A ____ blocks enemy attacks.` |
| 选项 | `shield` / `sword` / `water` / `book` |
| 正确答案 | `shield` |

### 答对反馈

```text
园丁：没错，shield 可以挡住攻击。
园丁：有些力量不是为了打倒别人，而是为了保护身边的人。
园丁：这页书也回来了。
```

### 答错反馈

```text
园丁：这个词还挡不住危险。
园丁：想一想，哪个东西可以挡住攻击？
```

### 奖励

- 书页 +1。
- 初心护符、盾牌或防御成长奖励。
- 后续可加入药水奖励，把 `water` 接到恢复系统。

## 森林入口与 Word Imp

### 未集齐三页时

森林入口应给出明确反馈：

```text
封印还没有完全松动。
《万语之书》还缺少 {missing_count} 页。
先去帮助村里的其他人吧。
```

### 集齐三页后

```text
《万语之书》发出温暖的光。
三页书重新合在一起，森林入口的封印开始松动。
一个小小的词灵从门后跳了出来。
```

### Word Imp 登场

```text
Word Imp：book？sword？shield？
Word Imp：我不记得了……我是谁？
《万语之书》：它不是坏孩子，只是忘记了自己的真名。
《万语之书》：准备好后，挑战它。让它安静下来，我会把它带回书里。
```

### 战斗规则叙事

战斗不再弹出单词答题卡。此处只通过叙事解释“之前学习带来的成长”：

```text
你握紧了 sword 唤醒的剑。
shield 的光在你身前亮起。
book 的书页等待记录迷失的词灵。
```

### 胜利结算文案

```text
你获得了胜利，击败了 Word Imp。
Word Imp 慢慢安静下来，变回一枚小小的词灵印记。
《万语之书》：第一只词灵回来了。
《万语之书》：但词灵王仍然不见踪影。森林深处，也许有下一条线索。
```

### 失败反馈

第一章不建议设置强惩罚。

```text
你暂时退回了村庄。
《万语之书》：没关系，见习勇者也需要练习。
《万语之书》：检查装备和生命值，再挑战一次吧。
```

## 第一章完成状态

完成后应满足：

- 书页显示 `3/3`。
- Word Imp 被击败。
- 森林入口打开。
- 玩家获得金币、经验和至少一个可见奖励。
- 《万语之书》提示：词灵王失踪，下一站是低语森林。

## 素材需求

### 已覆盖

- 主角战士：Tiny Swords warrior。
- 图书管理员：Tiny Swords monk。
- 铁匠：Tiny Swords pawn hammer。
- 园丁：Tiny Swords pawn wood。
- Word Imp 当前可用 Skull 敌人替代。
- 村庄建筑、树、石头、南瓜、稻草人、路牌、栅栏。
- 像素 UI 面板、按钮、血条、图标。

### 建议补充

- 《万语之书》专属图标或小型书本立绘。
- 书页飞回书中的小特效。
- Word Imp 更贴合“词灵”的专属外观，避免长期使用骷髅替代。
- `book / sword / shield` 三个核心词对应的更准确物品图标。

### 暂不需要

- 独立过场动画。
- 大量新怪物。
- 大量城市场景。
- 学校主题素材。
- 日语、德语文化道具。

| 剧本元素 | 当前实现入口 | 备注 |

## 测试与截图需求

### 行为测试

- 三个 NPC 支线完成后书页为 `3/3`。
- 答题结果通过 NPC 下一轮对话表达。
- Word Imp 只在 `3/3` 后可挑战。
- 战斗胜利后森林入口打开。
- 奖励进入 HUD、背包或装备系统。

### 视觉截图

- 图书管理员对话。
- 拼写挑战。
- 语境挑战。
- Word Imp 登场。
- 战斗聚焦蒙版。
- 战斗结算。
- 第一章完成后森林入口开启。
